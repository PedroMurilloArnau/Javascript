FUNCIONES

Conceptos basicos

Las funciones son bloques de codigo diseñados para realizar una determinada tarea. Y funcionan al ser invocadas

Sintaxis basica

Las funciones se invoca con la palabra reservada function seguido del nombre de la funcion y terminado con ()


Dentro de los parentesis podemos incluir los parametros que le pasamos dicha función


El codigo a ejecutiar se escribira a continuacion entre dos {}

Dentro de la funcion los parametros funcionan como variables locales


Para que usar Funciones

Usamos funciones principalmente por 3 razones

  1. Con las funciones podemos reusar el codigo
  2. Podemos escribir codigo usado muchas veces
  3. Podemos usar el codigo con distintos argumentos obteninendo distintos resultados
