Palabras reservadas

  1. var -- Declaramos una variable.
  2. let -- Declaramos un bloque de variable.
  3. const -- Declaramos una constante.
  4. if -- ejecutamos bajo una condicion.
  5. switch -- ejecutaremos bajo unos determinados casos.
  6. for -- ejecutaremos en loop.
  7. function -- declaramos una funcion.
  8. return -- devolvemos el valor de la funcion.
  9. try -- controlamos el manejo de errores en un bloque de codigo.

Variable "let"

A continuacion vamos a ver como funciona la variable "let" en JavasCript.

La variable "let" tiene las siguientes caracteristicas:

  1. Las variables definidas como "let" no se pueden volver a declarar.
  2. Debes de declararla antes de usarla.
  3. Las variables definidas como "let" tienen un alcance de bloque.


Variable "var"

Vamos a ver las diferentes características de la variable "var":

  1. Las variables "var" se pueden redefinir.
  2. Las variables "var" se pueden declararlas despues de definirlas.

Constantes

Las constantes tienen las siguientes características:

  1. Las constantes no pueden ser redeclaradas.
  2. Las constantes no puenden ser redefinidas.
  3. Las constantes tienen un alcance de bloque.
