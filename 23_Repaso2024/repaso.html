REPASO datos

Los datos es algo fundamental y trabajar con ellos de forma correcta es fundamental, ya que de ellos depende muchas veces el buen funcionamento de lo que queramos programar

Tipo de dato Explicacion Ejempo
String Se escriben entrecomilladas "Hola mundo"
Number Decimales o no 10.5
BigInt Guardan numeros demasiado grandes BigInt("654484646456454454621654317521336")
Booleans Puede ser dos valores. O "true" o "false" true
Arrays Almacena listas de datos const animales = ["Gato","Perro","Cisne","Ganso"]
Objects Podemos almacenar objetos const persona = {nombre:"Pedro",apellido:"Murillo",edad:30}
Undefined Una variable con un valor no definido let altura; arrojaria "undefined"
Empty Values Es un string vacio const nombre=""; Arroja un valor vacio de tipo string

Para conocer el tipo de valor que obctenemos podemos utilizar typeof
